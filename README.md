La especificación del lenguaje se encuentra en **doc**

Para compilar:

    cd src
    make

El ejecutable estará en src/devanix

Para ejecutar, hacer:

    devanix archivo

En el directorio **examples** hay varios ejemplos, algunos erróneos a propósito
para probar el compilador.
